# center-message

#### Description
推送中台 websocket渠道部分，基于socketio实现的websocket服务端，websocket服务端又实现了client和server，当客户端信息不在本机时，通过缓存定位到所在的服务器，然后进行Rpc通信，完成websocket服务端之间的消息路由转发。本项目包含websocket集群的基本逻辑和基于netty手写Rpc通信。适合学习。
